# /optimize - MANA Optimization Command

Use MANA's optimized tools for this task. Replace native tools with their MANA equivalents:

- Use `read_optimized` instead of `Read` for file reading
- Use `bash_optimized` instead of `Bash` for command execution
- Use `search_optimized` instead of `Glob`/`Grep` for searching
- Use `list_directory_optimized` instead of `Glob` for directory listing

Apply the EXTRACT action type when looking for specific code elements.
Apply the SUMMARIZE action type when understanding overall structure.
Apply the CHECK action type before making edits.
